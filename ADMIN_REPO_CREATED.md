## 🚀 Next Steps

### 1. View Your Repository
Visit: https://github.com/Geraldsaviour/lekki-stays-admin

### 2. Clone the Repository (Optional)
```bash
git clone https://github.com/Geraldsaviour/lekki-stays-admin.git
cd lekki-stays-admin
```

### 3. Enable Firebase Authentication
1. Go to: https://console.firebase.google.com/project/lekki-stays/authentication
2. Enable Email/Password authentication

### 4. Create Admin User
1. In Firebase Console → Authentication → Users
2. Add user: admin@lekkistays.com

### 5. Update Bank Details
1. Edit `src/js/dashboard.js` (line ~280)
2. Update bank information

### 6. Deploy to Firebase Hosting
```bash
cd lekki-stays-admin
npm install
firebase login
firebase init hosting
firebase deploy --only hosting
```

## 🎯 Key Features of the Message

### ✅ **Clear Status**
- 🚫 Emoji immediately shows it's a cancellation
- Bold "BOOKING CANCELLED" header
- Clear statement of who cancelled (guest or host)

### ✅ **Complete Information**
- Booking reference for tracking
- Property name and location
- Original dates and guest count
- Number of nights

### ✅ **Transparent Refund Details**
- Original amount paid
- Refund amount (if any)
- Processing timeline (5-7 days)
- Refund method
- Clear reason if no refund

### ✅ **Cancellation Reason** (when applicable)
- Only shown for host cancellations
- Explains why booking was cancelled
- Professional and respectful

### ✅ **Appropriate Tone**
- **Guest cancellation:** Neutral, professional, welcoming
- **Host cancellation:** Apologetic, empathetic, helpful

### ✅ **Next Steps**
- Clear call-to-action
- Link to browse properties
- Contact information
- Invitation to book again

### ✅ **Support Access**
- Direct reply option
- Phone number provided
- Encourages questions

## 🔄 Guest Can Reply Directly

After receiving the message, the guest can:

1. **Reply with questions:**
   ```
   "When exactly will I receive the refund?"
   "Can I rebook for different dates?"
   "Why was my booking cancelled?"
   ```

2. **Click the website link:**
   - Opens in browser
   - Can browse alternative properties
   - Can make new booking

3. **Call the phone number:**
   - Direct call to support
   - Speak with team member
   - Get immediate assistance

4. **Save booking reference:**
   - Long-press to copy
   - Save for records
   - Use for future inquiries

## 🎨 Message Design Principles

### 1. **Scannable**
- Short paragraphs
- Clear sections with headers
- Bullet points for lists
- Emojis as visual markers

### 2. **Complete**
- All necessary information
- No need to ask follow-up questions
- Clear next steps
- Contact information

### 3. **Professional**
- Proper grammar and spelling
- Respectful tone
- No slang or casual language
- Brand-appropriate

### 4. **Mobile-Optimized**
- Short lines (40-50 characters)
- Adequate spacing
- Easy to read on small screens
- No horizontal scrolling

### 5. **Actionable**
- Clear call-to-action
- Clickable links
- Easy to respond
- Next steps obvious

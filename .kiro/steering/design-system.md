## Icon System

- ALWAYS use Lucide icons for all icons across the project
- NEVER use emojis as icons
- Import via CDN: https://unpkg.com/lucide@latest
- Initialize with: lucide.createIcons() at bottom of body
- Icon size: 20px default, 24px for feature sections
- Icon color: inherit from parent or use accent var(--accent)

### Amenity Icons Mapping
